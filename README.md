 Data Visualization and Storytelling - Task 2

## Objective
To create meaningful visualizations and extract business insights from sales data.

## Tools Used
- Power BI
- Excel Dataset (Superstore.csv)

## Dataset
Superstore sales dataset containing order, sales, profit, and customer data.

## Visualizations Created
1. Sales by Region
2. Monthly Sales Trend
3. Category-wise Sales
4. Profit by Segment
5. Top 10 Products
6. Sales by State

## Key Insights
- West region generates highest revenue.
- Technology category dominates sales.
- Sales peak during festive months.
- Corporate segment gives maximum profit.

## Conclusion
This dashboard helps understand business performance and supports data-driven decision making.
